# Personality Based Fortune Teller

A simple Python menu-driven application that predicts a user's personality based on their answers to a set of questions and generates a fortune message.

## Features
- Take a personality test using yes/no questions
- Generate personality type based on scores
- Display fortune message
- View latest result
- Search user by name
- Delete user record
- Input validation for user responses

## Technologies and Concepts Used
- Python 
- Functions
- Lists and Dictionaries
- Loops and Conditional Statements
